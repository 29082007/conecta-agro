# conecta-agro
O presente projeto tem como finalidade o desenvolvimento de uma plataforma digital (site) voltada à organização e disponibilização de currículos de estudantes do curso Técnico em Agropecuária da cidade de Uruguaiana/RS. A proposta visa aproximar os alunos do mercado de trabalho, facilitando o acesso a oportunidades de estágio e emprego no agro
